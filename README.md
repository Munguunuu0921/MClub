 хөгжүүлэх шилдэг систем болох" алсын хараатай MClub клубуудын нэгдсэн системийн хөгжүүлэлт юм. Энэ нь МУИС-ХШУИС-ийн "Хүн компьютерийн харилцаа" хичээлээр хийсэн төсөл юм.
